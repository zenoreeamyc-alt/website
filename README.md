# zass-clone

Clone-style landing page inspired by a profile/link hub.

## Install
```bash
npm install
```

## Dev
```bash
npm run dev
```

## Build
```bash
npm run build
npm run start
```

## Customize
Edit branding/links in `src/config/site.ts`.

Put images in:
- `public/images/cover.jpg`
- `public/images/avatar.jpg`
